## 6. Yapay Zeka Çıktıları, Üçüncü Taraf LLM İşleme ve İnsan Denetimi

### 6.1 Yapay Zeka Tarafından Üretilen İçerik

Hizmet, yapay zeka ile yanıtlar, özetler, puanlar ve öneriler üretir. Aşağıdakileri anladığınızı ve kabul ettiğinizi beyan edersiniz:

- Yapay zeka çıktıları hukuki, tıbbi, finansal veya diğer profesyonel tavsiyelerin yerine geçmez.

### 6.2 Üçüncü Taraf LLM İşleme

Yapay zeka özelliklerini sunabilmek için verileriniz üçüncü taraf büyük dil modeli (LLM) sağlayıcıları tarafından işlenir. Şu anda metin üretimi, çıkarım ve gömme (embedding) için **OpenAI (OpenAI, L.L.C., ABD)** kullanmaktayız.

Hizmeti kullanarak aşağıdakileri kabul etmiş olursunuz:

- Müşteri konuşma içeriklerinin işlenmek üzere OpenAI'ın **Amerika Birleşik Devletleri**'ndeki API sunucularına iletilmesi.
- Kişi çıkarımı ve puanlamanın üçüncü taraf yapay zeka çıkarımı yoluyla gerçekleştirilmesi.
- Bilgi bankası içeriğinin üçüncü taraf yapay zeka hizmetleri aracılığıyla gömülmesi (embedding) ve erişilmesi.

OpenAI'ın API hizmetini kullanıyoruz. OpenAI'ın API veri kullanım politikasına göre API verileri modellerini eğitmek için kullanılmaz.
